# Technical Stack

## Frontend

- Next.js with the App Router.
- TypeScript for strict typing and maintainability.
- React Server Components where beneficial.
- Framer Motion for premium but controlled motion.

## Styling and UI Foundation

- Design tokens and CSS variables for theme control.
- A component system built for consistency and scale.
- Dark mode as a first-class system, not an afterthought.

## Content and Data

- Curated content modules for products, pages, and FAQs.
- Optional headless CMS if the team needs editor-friendly publishing later.
- Structured metadata for SEO and social sharing.

## Media and Performance

- Optimized responsive images.
- Lazy loading for below-the-fold media.
- Minimal client-side JavaScript.
- Progressive enhancement for all key flows.

## Forms and Lead Capture

- Enquiry forms with clear intent options.
- Email routing or CRM integration.
- Spam protection and form validation.
- Conversion analytics on all primary CTAs.

## Observability

- Analytics for scroll depth, CTA clicks, and form submits.
- Search console monitoring.
- Performance budgets and Core Web Vitals tracking.

## Hosting and Deployment

- Modern edge-ready hosting.
- Preview environments for content and design review.
- Automated deployment with rollback safety.

## Stack Decision Principle

Choose technologies that make the site faster, easier to maintain, and easier to scale without sacrificing the premium experience.
