# Design System Structure

## Design Principles

- Premium before decorative.
- Calm before loud.
- Editorial spacing before dense composition.
- Trust before novelty.

## Color Strategy

- Base palette should feel warm, natural, and sophisticated.
- Use restrained neutrals with one strong accent only where it supports action.
- Support dark mode with a separate tonal system rather than an inverted afterthought.

## Typography Strategy

- Use refined, high-legibility typography with a strong editorial presence.
- Establish a clear scale for hero, section, and supporting copy.
- Keep line length disciplined for luxury reading comfort.

## Spacing and Layout

- Large whitespace is intentional, not empty.
- Use a consistent rhythm across sections.
- Maintain strong alignment and generous margins on desktop.

## UI Language

- Lines, borders, and cards should be subtle.
- Elevation should be minimal and purposeful.
- Avoid over-stylized gradients or decorative clutter.

## Token Families

- Color tokens
- Typography tokens
- Spacing tokens
- Radius tokens
- Shadow or elevation tokens
- Motion tokens

## Accessibility Commitments

- Strong contrast across both themes.
- Visible focus states.
- Semantic hierarchy in all pages.
- Touch-friendly interaction sizes on mobile.
