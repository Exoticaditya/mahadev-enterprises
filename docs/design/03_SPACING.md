# 03 Spacing

## Spacing Philosophy

Space is part of the premium experience. The interface must breathe. If a layout feels crowded, it is failing the brand.

## 8pt Grid

All spacing must be based on an 8pt system with controlled intermediate steps only where necessary.

- 4 px for micro offsets and fine alignment.
- 8 px for tight internal spacing.
- 16 px for standard component rhythm.
- 24 px for comfortable separation.
- 32 px for section sub-spacing.
- 40 px for large component spacing.
- 48 px for major section padding.
- 64 px and above for spacious editorial breaks.

## Containers

- Desktop content width should feel generous but not overextended.
- Use a centered maximum width for editorial sections.
- Allow wider media and product grids where the layout benefits from breadth.
- Keep text-heavy sections narrower than media-heavy sections.

## Margins

- Outer margins must increase as the viewport widens.
- Mobile margins should never feel edge-to-edge unless intentionally immersive.
- Avoid inconsistent left and right breathing room.
- Maintain equal visual pressure on both sides of a section.

## Padding

- Cards should have enough internal padding to feel tactile.
- Forms should never feel compressed.
- CTA blocks should use generous padding to create importance.
- Hero sections should allow the headline and supporting copy to breathe.

## Whitespace Rules

- Whitespace is intentional structure, not emptiness.
- Large whitespace should mark transitions between major idea groups.
- Product content must not be crammed into a single dense fold.
- Editorial sections should feel paced like a luxury magazine.

## Responsive Spacing

- Desktop: more vertical air, more section separation, broader horizontal padding.
- Tablet: preserve hierarchy while slightly tightening rhythm.
- Mobile: reduce total scale, not dignity; keep generous line spacing and usable touch space.
- Spacing should compress gracefully without collapsing the premium feel.

## Spacing Rules

- Never use arbitrary spacing values without reason.
- Keep component spacing consistent across all page types.
- Use larger gaps to separate ideas, smaller gaps to group related content.
- Never let a section feel like a dump of information.
